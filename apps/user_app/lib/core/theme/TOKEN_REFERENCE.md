## ðŸ“‹ Token Cheat Sheet

### Backgrounds
- `bgApp(context)` - Main app background
- `bgSurface(context)` - Cards, surfaces
- `bgElevated(context)` - Dialogs, modals
- `bgInput(context)` - Input fields

### Text
- `textPrimary(context)` - Headings
- `textSecondary(context)` - Body text
- `textTertiary(context)` - Hints
- `textDisabled(context)` - Disabled

### Icons
- `iconPrimary(context)` - Main icons
- `iconSecondary(context)` - Secondary icons
- `iconTertiary(context)` - De-emphasized

### Borders
- `borderDefault(context)` - Default borders
- `borderInput(context)` - Input borders
- `borderInputFocused(context)` - Focused inputs

### Actions
- `actionPrimary(context)` - Primary buttons
- `actionSecondary(context)` - Secondary buttons
- `actionPrimaryText(context)` - Text on primary
- `actionSecondaryText(context)` - Text on secondary

### States (Unchanged)
- `stateSuccess` - Success
- `stateError` - Error
- `stateWarning` - Warning
- `stateInfo` - Info

### Brand (Unchanged)
- `brandPrimary` - Yellow (#FDBB42)
- `brandSecondary` - Red (#9C100E)
- `brandAccent` - Peach (#FFE8D6)
